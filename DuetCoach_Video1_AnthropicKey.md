# Video 1: Getting Your Anthropic API Key
## Target length: 3-4 minutes

---

## Loom Setup
- Record desktop, your face in corner
- Have two browser tabs ready: console.anthropic.com and DuetCoach Settings page
- Sign out of both before recording so you can show the full flow

---

## Script

### [0:00] Intro
"Hey, this is Steve. This video shows you how to get your Anthropic API key — that's the one that powers the Claude AI conversation mode in DuetCoach. It takes about 3 minutes, costs nothing to set up, and your typical usage will run about $3-5 a month depending on how often you practice.

Important: this key stays in your own browser. DuetCoach never sees it. You're paying Anthropic directly for what you use."

### [0:20] Step 1 — Go to Anthropic Console
"First, open a new tab and go to **console.anthropic.com**. That's c-o-n-s-o-l-e dot anthropic dot com.

If you have a Google account, just click 'Continue with Google' — that's the fastest way. Otherwise, sign up with your email."

[Show: clicking Continue with Google, completing signup]

### [0:50] Step 2 — Add Credit
"Once you're in, you'll see your dashboard. Anthropic charges as you use it, so you need to add some credit to your account first. Click on the 'Plans & Billing' or 'Billing' section.

I usually start with a $5 credit — that lasts me about a month of daily practice. You can always add more later."

[Show: navigating to billing, adding $5 credit, entering card]

### [1:30] Step 3 — Create the API Key
"Now click on 'API Keys' in the left sidebar — or go directly to console.anthropic.com/settings/keys.

Click the **'Create Key'** button.

Give it a name so you remember what it's for. I'll call mine 'DuetCoach.'

Set workspace to 'Default' — that's fine.

Click 'Create Key.'"

[Show: API Keys page, Create Key button, naming the key, clicking Create]

### [2:00] Step 4 — Copy the Key
"This is the most important part. The key will appear ONCE. After you close this dialog, you can't see it again — you'd have to create a new one.

Click the copy button next to the key. It starts with **sk-ant-** followed by a long string of letters and numbers."

[Show: the key, clicking copy]

### [2:20] Step 5 — Paste Into DuetCoach
"Now hop over to DuetCoach. Sign in if you haven't already.

Click the **Settings** tab.

Find the **'Claude API Key'** section.

Paste your key into the box.

Click **Save.**

Click **Test** — you should see a green status dot. That means it's working."

[Show: pasting the key, clicking Save, clicking Test, seeing the green light]

### [2:50] Wrap Up
"You're done with the Anthropic key. Now you can use Claude AI mode and Script Training mode in DuetCoach. Next video, I'll show you how to get the ElevenLabs key for voice mode. Talk to you in a minute."

## Things to Remember to Show
- Where to copy the API key from
- The exact Settings location in DuetCoach
- That the Test button confirms it works (green dot)
- Mention: never share your key, never email it, never paste it into a public site

## Things NOT to Show
- Your actual API key (use a fake one or blur it in editing)
- Your billing/credit card info
